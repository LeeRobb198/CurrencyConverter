Purpose of iCurrency


The application has four functions within it. The user can compare 32 different currencies live by either standard numerals, a bar chart or a line chart. The application has the ability for the user to:
  1. Compare the exchange rate of one currency against all currencies presented in a numerical form.

  2. Convert a currency given a value of one currency converted to the amount received of each other currencies.

  3. Compare the exchange rate of one currency against all currencies presented in a bar chart form.

  4. Compare historical exchange rates between two dates comparing one currency against another.
Comparing the exchange rate of one currency against another, presenting the data in a numerical form requires the user to select a currency from a dropdown of options. Finding the value received given a value of a chosen currency requires the user to enter a numerical value and the currency they wish to convert. To compare the exchange rate of a currency against all currencies presenting the data in a bar chart form requires the user to select a currency from a dropdown of options in addition to setting a minimum and maximum limit. This will scale the graph down to limit the number of currencies present. For example, selecting GBP and setting a limit between 0 â€“ 20 will not display currencies where the value of 1 GBP would give a currency value back of anything > 20. The final function of comparing historical data requires the user to select a currency and a currency they wish to compare against from two separate dropdowns. Also, the user will need to choose two dates to compare between.
